# 校园导航演示 (Campus Navigation Demo)

这是一个基于C++的控制台应用程序，旨在模拟和解决校园内的各种导航问题。该项目将校园地图抽象为一个图（Graph），并实现了一系列复杂的图算法，以提供丰富的路径规划和分析功能。

## 主要功能

本项目通过一个交互式菜单，为用户提供以下核心功能：

1.  **地图信息展示**：显示校园中所有地点和道路（边）的信息，包括道路长度（权重）。
2.  **动态地图编辑**：
      * **增删地点**：允许用户在地图上动态添加或删除地点（顶点）。
      * **增删道路**：允许用户在地点之间动态添加或删除道路（边）。
3.  **最短路径查询**：
      * **两点间最短路径**：计算并显示任意两个指定地点之间的最短路径及其长度。该功能通过 **Dijkstra算法** 实现。
4.  **复杂路径规划**：
      * **经过N个地点的最短路径**：查询从起点到终点，且路径恰好包含N个地点的最短路线。
      * **关键地点导览**：为一组必须访问的“关键地点”，规划出一条总长度最短的游览路径。此功能首先通过 **Prim算法** 生成最小生成树以简化网络，然后通过 **全排列（Brute-force TSP）** 的方式在关键地点间寻找最优顺序。
      * **限制游览顺序的路径**：在给定一系列“先后访问”的限制条件（例如，必须先到A点才能到B点）下，寻找一条满足所有限制条件的最短路径。该功能通过深度优先搜索找出所有可行路径，再进行筛选。

## 核心技术与算法

项目的核心是一个泛型 `Graph` 类，它使用邻接表来存储图结构，并实现了多种高级图算法。

  * **图数据结构**：

      * 使用 `std::map<T, std::set<Edge<T>>>` 作为邻接表，其中 `T` 可以是地点名称（`std::string`）或其整数ID (`int`)。 `Edge` 类封装了相邻顶点和边的权重。

  * **实现的算法**：

      * **最短路径算法**：
          * **Dijkstra算法**：用于解决单源最短路径问题，是查询两点间最短路径的核心。
          * **Floyd-Warshall算法**：用于计算图中所有顶点对之间的最短路径，在处理多关键地点的导览路径时被调用。
      * **最小生成树 (MST)**：
          * **Prim算法**：用于寻找图的最小生成树，为“最优导览路径”功能提供一个简化的道路网络。
      * **图遍历算法**：
          * **深度优先搜索 (DFS)** 和 **广度优先搜索 (BFS)**：用于图的遍历和寻找特定属性的路径。
      * **拓扑排序**：
          * 实现了拓扑排序算法，可用于检测图中是否存在环，确保路径规划的有效性。
      * **图连通性分析**：
          * **连通分量**：能够识别并输出图中的所有连通分量。
          * **关节点 (Articulation Points)**：能够通过暴力法和 **Tarjan算法** 两种方式找到图中的关节点（割点），用于分析网络中的关键位置。

## 项目结构

  * `main.cpp`: 程序主入口，负责创建图对象并处理用户的菜单输入。
  * `graph.hpp`: 核心文件，定义了 `Graph` 模板类及所有图论算法的实现。
  * `edge.hpp`: 定义了 `Edge` 类，用于表示图的带权边。
  * `create_map.cpp`: 负责初始化校园地图的默认数据，包括地点和道路。
  * `map_ope.cpp`: 实现了 `main.cpp` 中定义的各项用户操作的具体逻辑。

## 如何编译与运行

你可以使用支持C++11或更高版本的编译器来编译本项目。

```bash
# 使用 g++ 编译器
g++ main.cpp -o CampusNavigationDemo -std=c++11

# 运行程序
./CampusNavigationDemo
```

程序启动后，将显示一个操作菜单。根据提示输入数字编号即可使用相应的功能。
